Confidentialité

La confidentialité compte quand les clients uploadent des photos

Les photos servent uniquement à générer l'aperçu d'essayage. Nous ne stockons pas les photos originales dans notre base.

Que deviennent les photos clients

  • Traitées de façon sécurisée pour générer l'aperçu
  • Les photos originales ne sont pas enregistrées dans la base Stylab après traitement
  • Transitent via notre fournisseur IA (Replicate) pour l'inférence uniquement
  • Le widget affiche clairement « photo non stockée » avant l'upload

Ce que nous conservons pour votre boutique

  • Événements d'usage (sessions, taux de succès, produit consulté)
  • URLs d'aperçu dans les logs serveur (liens Replicate expirent ~1 h)
  • Vos photos flat-lay dans Shopify Files
