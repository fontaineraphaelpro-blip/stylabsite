Storefront · widget

The widget on every product page

A 5,000+ line Shadow DOM widget engineered for 100% Shopify theme coverage.

  • Closed Shadow DOM — zero CSS conflicts with the theme.
  • 9-layer DOM anchor: custom selector → product form + ATC → global ATC → product-form element → product-info containers (Dawn / PageFly / GemPages / Shogun + 25 more) → embed slot → sticky ATC bar → main → floating fallback.
  • MutationObserver waits up to 12 seconds for late-rendered sections.
  • Boot retries at 800ms, 2s, 4.5s, 9s, 15s, 22s + Shopify section reload hooks.
  • Mounts only on /products/* — never on cart, search, header or drawers.
  • Self-heals if a theme removes the container (re-injects up to 6 times).

Privacy & trust

The shortest privacy story in the category.

Shopper photos never touch persistent storage. We log result URLs and operational data only. Three Shopify GDPR webhooks are implemented and verifiable in the codebase.

What we store

  • Result image URL hosted on Replicate CDN.
  • Try-on log metadata: shop, product, customer ID or IP, latency, success / failure, error.
  • Anonymous visitor ID in shopper's localStorage for A/B bucketing.
  • Custom garment images uploaded by merchants — stored in Shopify Files.

What we never store

  • The shopper's uploaded photo — sent in-memory to the AI engine and discarded.
  • The shopper's name, address, or payment info — we don't ask, we don't see.
  • The garment image alongside the user photo after generation completes.
  • Identifying data in GDPR data-request exports — only log metadata.
